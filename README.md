# Простая пересменка

Автоматизируйте утреннюю пересменку за несколько кликов.  
Программа проверит гарнитуру, микрофон (и опционально камеру), откроет нужные документы и подключит вас к Яндекс.Телемосту — останется только представиться.

---

## 📋 Возможности

- 🎧 **Проверка наушников** – тихий тональный сигнал, чтобы не мешать коллегам.
- 🎤 **Проверка микрофона** – слышите свой голос в реальном времени через гарнитуру.
- 📷 **Проверка веб-камеры** (включена по умолчанию) – окно с живым видео; при закрытии тест останавливается корректно.
- 📄 **Автоматическое открытие документов Word** (опционально):
  - Поиск доклада и справки по сетевым папкам согласно дате.
  - Утренняя логика: если запуск до 12:00, ищет документы за вчерашний день.
  - Резервный поиск справки по дате в альтернативном расположении.
  - Размещение окон на выбранном мониторе (левый/правый).
  - Если найден только один документ – открывается на весь экран выбранного монитора.
- ⚡ **Быстрый старт** – возможность пропустить все проверки и сразу перейти к документам и Телемосту.
  - Кнопка «⚡ Быстрый старт» на первом экране (показывается опционально).
  - Режим полного автомата: при `"skip_tests": true` программа вообще не показывает окон, сразу выполняя свою работу.
- 📞 **Мгновенный вход в Телемост** – программа находит приложение Яндекс.Телемост автоматически (через реестр или стандартные пути) и подключается к конференции без браузера.
- ⚙️ **Гибкая настройка** – внешний `config.json` позволяет менять всё: от ссылки на встречу до включения быстрого старта.
- 🧩 **Поддержка нескольких конфигов** – через аргумент `--config` можно указать любой JSON-файл с настройками. Удобно для разных ролей (оператор, менеджер и т.д.).
- 💻 **Готовый EXE** – не требует установки Python; все зависимости уже внутри.
- 🛡️ **Понятные ошибки** – при повреждённом `config.json`, отсутствии микрофона или недоступности файлов программа показывает дружелюбное предупреждение и продолжает работу с безопасными настройками.

## 🚀 Быстрый старт (для конечного пользователя)

1. Скачайте `Peresmenka.exe` и `config.json` из [раздела Releases](https://github.com/yourusername/yourrepo/releases).
2. Положите оба файла в одну папку (например, `C:\ProgramData\Peresmenka`).
3. Отредактируйте `config.json` в Блокноте, указав актуальную ссылку на Телемост и другие параметры.
4. Запустите `Peresmenka.exe`. При необходимости создайте ярлык на рабочем столе.
5. Чтобы программа запускалась по расписанию (будние дни в 8:10), настройте задачу в Планировщике задач Windows (см. раздел «Автозапуск»).

> 💡 **Совет:** если вы хотите использовать разные настройки для разных сотрудников, создайте несколько конфигов (например, `config_operator.json`, `config_manager.json`) и запускайте программу с ключом `--config` (см. ниже).

## ⚙️ Настройка config.json

Все параметры собраны в одном файле. Если файл отсутствует, используются значения по умолчанию (камера включена, документы выключены).

| Ключ | Тип | Описание |
|------|-----|----------|
| `telemost_url` | string | Полная ссылка на конференцию (например, `https://telemost.yandex.ru/j/1234567890`) |
| `enable_camera_check` | bool | Показать кнопку «Проверить камеру» (`true` / `false`). По умолчанию `true` |
| `enable_documents` | bool | Открывать документы Word (`true` / `false`). По умолчанию `false` |
| `network_base_path` | string | Сетевой путь к основной папке (только если `enable_documents: true`) |
| `reserve_base_path` | string | Сетевой путь к резервной папке (только если `enable_documents: true`) |
| `doklad_keyword` | string | Ключевое слово для поиска файла доклада |
| `spravka_keyword` | string | Ключевое слово для поиска файла справки |
| `preferred_monitor` | int | Номер монитора для размещения окон Word (0 – главный, 1 – второй и т.д.) |
| `main_path_template` | string | Шаблон основного пути к папке с документами (см. ниже) |
| `reserve_path_template` | string | Шаблон резервного пути (см. ниже) |
| `skip_tests` | bool | Полностью автоматический режим: `true` – программа не показывает интерфейс и сразу выполняет открытие документов и Телемоста. По умолчанию `false` |
| `show_quick_start_button` | bool | Показывать кнопку «⚡ Быстрый старт» на первом экране (`true` / `false`). Работает только при `skip_tests: false`. По умолчанию `true` |

### 🧩 Шаблоны путей (только для `enable_documents: true`)

С помощью шаблонов вы можете гибко описать, где лежат ваши документы, не меняя код программы.  
Шаблон – это строка, в которую можно вставлять переменные, заключённые в фигурные скобки. Программа автоматически заменит их на актуальные значения.

**Доступные переменные:**

| Переменная | Что подставится | Пример (на 29 мая 2026) |
|------------|-----------------|-------------------------|
| `{network_base}` | Основной сетевой путь (`network_base_path`) | `\\server\share` |
| `{reserve_base}` | Резервный сетевой путь (`reserve_base_path`) | `\\server\archive` |
| `{year}` | Год (четыре цифры) | `2026` |
| `{month}` | Номер месяца (две цифры) | `05` |
| `{month_name}` | Название месяца (с маленькой буквы) | `май` |
| `{search_date_short}` | Дата поиска в формате ДД.ММ.ГГ (если до полудня – вчерашняя) | `28.05.26` |
| `{date_short}` | Текущая дата в формате ДД.ММ.ГГ | `29.05.26` |
| `{date_full}` | Текущая дата в формате ДД.ММ.ГГГГ | `29.05.2026` |

**Примеры шаблонов:**

1. **Простая структура:** `\\server\reports\2026\05 май\28.05.26\Доклады`  
   Шаблон:  
   `{network_base}\{year}\{month} {month_name}\{search_date_short}\Доклады`

2. **Резервный путь без даты:** `\\server\archive\2026\05 май`  
   Шаблон:  
   `{reserve_base}\{year}\{month} {month_name}`

3. **Фиксированная папка вне даты:** `\\server\docs\Ежедневно`  
   Шаблон:  
   `\\server\docs\Ежедневно` (просто строка без переменных)

Если шаблоны не заданы (пустые строки), программа будет использовать старый способ:  
`{network_base}\{year}\{month} {month_name}\{search_date_short}\ДЧ` для основного пути и  
`{reserve_base}\{year}\{month} {month_name}` для резервного.

**Внимание:**  
- Обратные слеши в JSON необходимо дублировать (`\\\\`).  
- Вместо `\\\\server\\share` можно использовать прямой слеш: `//server/share`.

### 📄 Пример `config.json`

    {
        "telemost_url": "https://telemost.yandex.ru/j/12345678901234",
        "enable_camera_check": true,
        "enable_documents": true,
        "network_base_path": "\\\\server\\share",
        "reserve_base_path": "\\\\server\\archive",
        "doklad_keyword": "Доклад",
        "spravka_keyword": "Справка",
        "preferred_monitor": 1,
        "main_path_template": "{network_base}\\{year}\\{month} {month_name}\\{search_date_short}\\Доклады",
        "reserve_path_template": "{reserve_base}\\{year}\\{month} {month_name}",
        "skip_tests": false,
        "show_quick_start_button": true
    }

### 🔄 Запуск с другим конфигом

Вы можете указать программе использовать любой JSON-файл с настройками через аргумент командной строки `--config`:

    Peresmenka.exe --config config_operator.json

Это позволяет создать несколько конфигов для разных ролей и запускать их через отдельные ярлыки.  
Файл конфига должен находиться в той же папке, что и программа, или можно указать полный путь:

Peresmenka.exe --config "C:\Users\Ivanov\Documents\my_config.json"

Если аргумент не передан, используется стандартный `config.json`.

## 🛠 Разработка и сборка из исходников

### Требования

- Python 3.9+
- Установленные библиотеки (см. `requirements.txt`)

### Установка зависимостей

    pip install -r requirements.txt

### Сборка EXE

    pyinstaller --onefile --windowed --name "Простая_пересменка" --icon=Peresmenka.ico --hidden-import=cv2 --collect-all opencv-python main.py

После сборки исполняемый файл появится в папке dist.

### Запуск без компиляции

    python main.py

Программа также прочитает config.json, лежащий рядом с main.py.

## 🗓 Автозапуск (планировщик задач)

Чтобы программа запускалась автоматически по будням в заданное время, создайте простую задачу в Планировщике задач Windows:

1. Откройте `taskschd.msc`.
2. Создайте задачу с триггером «Еженедельно» (пн–пт) и действием «Запустить программу» – укажите путь к `Peresmenka.exe`.
3. На вкладке «Условия» снимите галочку «Останавливать при переходе на питание от батарей» (если нужно).
4. Готово. Теперь программа будет сама напоминать о пересменке.

Готовые скрипты `.bat` / `.ps1` не включены в репозиторий, чтобы каждый мог настроить задачу под свои требования.
